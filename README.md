## Project Description
The Sales Data Analysis Dashboard is a Python-based application that analyzes sales data and generates meaningful business insights. The project loads sales data from CSV files, cleans and preprocesses the data, performs exploratory data analysis, calculates key metrics, creates visualizations, and exports reports. It demonstrates real-world data analysis workflows using Python and pandas.

---

## Project Objectives
- Load sales data from CSV/Excel files  
- Clean and preprocess raw data  
- Perform exploratory data analysis (EDA)  
- Calculate key sales metrics  
- Analyze trends and patterns over time  
- Generate visual and Excel-based reports
